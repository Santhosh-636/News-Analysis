## 📝 Configuration Guide

### Adding More News Sources
Edit `news_scraper.py`, Line: News sources dictionary
```python
self.news_sources = {
    'Your Source Name': 'RSS_FEED_URL',
    # Add more sources here
}
```

### Customizing Colors
Edit `dashboard.py`, Styling section:
```python
color_discrete_map={
    'Happy/Positive': '#YOUR_COLOR',
    # Update colors here
}
```

### Adjusting Language Detection
Edit `sentiment_analyzer.py`:
```python
lang_map = {
    'xx': 'Language Name',
    # Add more language codes
}
```
